# Code of Conduct

## Our Standard

We expect all participants to:

- communicate respectfully and directly
- focus on technical clarity and evidence
- accept constructive feedback
- avoid personal attacks and harassment

## Unacceptable Behavior

- insults, threats, or harassment
- discriminatory language
- deliberate disruption or bad-faith trolling
- publishing private information without consent

## Scope

This applies to:

- repository issues and pull requests
- discussions and review comments
- community spaces tied to this project

## Enforcement

Project maintainers may:

- request behavior changes
- remove abusive content
- limit or revoke participation for repeated violations

## Reporting

If you experience or witness unacceptable behavior, report it privately to maintainers with:

- links/screenshots
- timeline
- involved accounts

Reports will be reviewed and handled with discretion.
